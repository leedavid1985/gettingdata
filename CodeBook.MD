ID Fields
=====
*activityID* - Denotes activity
 *1 = Walking
 *2 = Walking upstairs
 *3 = Walking downstairs
 *4 = Sitting
 *5 = Standing
 *6 = Laying
*activityType* - Natural language description of activity
*subjectID* - Participant ID
